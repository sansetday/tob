◆ prog()

def extractip.prog ( )

Первый поток - обработка файлов.

Ищет входящие файлы и запускает еще шесть потоков для первых 1000 файлов. Добавляет результаты в очередь и исключает повторные IPs. Обработка следующих 1000 файлов. Если нет входящих файлов, то ждем появления элементов.
